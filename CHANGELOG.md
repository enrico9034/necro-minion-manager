# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-08

### Added
- Initial release of Grimorio del Necromante
- Creature management (add, edit, delete)
- HP tracking with heal/damage buttons
- Full creature stats display (AC, Speed, Attacks, etc.)
- Inventory and equipment management
- Custom notes for each creature
- Dark theme UI with necromancer aesthetics
- PWA support with offline functionality
- Local storage persistence
- Settings page with data import/export
- Documentation page
- Responsive design for mobile and desktop
- GitHub Pages deployment support
